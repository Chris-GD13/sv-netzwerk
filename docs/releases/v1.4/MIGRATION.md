# Migration v1.3 → v1.4

v1.4 ersetzt die Kernkomponenten aus v1.3 durch kompatible, erweiterte Fassungen und migriert die Startseite vollständig auf `BaseLayout`.

Bestehende Fachseiten können anschließend schrittweise auf das neue System umgestellt werden. Die zentralen Klassen `sv-container`, `sv-section`, `sv-grid`, `sv-card` und `sv-button` bilden den verbindlichen UI-Unterbau.
