# Installation – SV-Netzwerk v1.4 Modern Website

1. ZIP entpacken.
2. Den vollständigen Inhalt in den lokalen Repository-Stamm `...\GitHub\sv-netzwerk\` kopieren.
3. Vorhandene Dateien ersetzen; übrige Bestandsdateien bleiben erhalten.
4. GitHub Desktop öffnen und Änderungen prüfen.
5. Commit-Nachricht: `SV-Netzwerk v1.4 Modern Website`.
6. Commit und Push ausführen.
7. GitHub Actions `Build Check` und produktiven Deploy kontrollieren.

## Lokale Prüfung

```bash
cd sv-netzwerk
npm install
npm run build
```
