# Installation – SV-Netzwerk v1.5

1. ZIP entpacken.
2. Den vollständigen Inhalt in den lokalen Repository-Stamm `...\GitHub\sv-netzwerk\` kopieren.
3. Vorhandene Dateien ersetzen.
4. GitHub Desktop öffnen und Änderungen prüfen.
5. Commit-Summary: `SV-Netzwerk v1.5 Knowledge & Navigation`
6. Commit und Push ausführen.
7. Build-Check und IONOS-Deploy in GitHub Actions prüfen.
