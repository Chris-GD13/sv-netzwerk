\# SV-Netzwerk



Offizielle Projektquelle der Website \*\*SV-Netzwerk\*\*.



\## Website



https://www.sv-netzwerk.eu



\---



\## Technik



\- Astro

\- Node.js 24

\- Git

\- GitHub

\- GitHub Actions

\- IONOS Deployment



\---



\## Installation



```bash

npm install

```



Entwicklungsserver



```bash

npm run dev

```



Build



```bash

npm run build

```



\---



\## Deployment



Jeder Commit auf den Branch \*\*main\*\* löst automatisch aus:



1\. GitHub Actions

2\. Build der Website

3\. Upload nach IONOS per SFTP
