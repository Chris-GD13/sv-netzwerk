# Benutzeranleitung fuer Pruefer

1. Ueber `/intern/login/` anmelden.
2. Im Dashboard Status, eigene Zuordnungen und letzte Aenderungen pruefen.
3. In der Fensterliste Datensatz oeffnen oder duplizieren.
4. Pflichtfelder mit Stern ausfuellen.
5. Fotos direkt ueber Smartphone oder Tablet hochladen.
6. Zwischenspeichern nutzt lokalen Entwurf und Supabase-Synchronisierung.
7. Vor `Pruefung abschliessen` zeigt das System eine Zusammenfassung.
8. Freigegebene Datensaetze sind fuer Pruefer schreibgeschuetzt.
