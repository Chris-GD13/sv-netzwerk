# Datenschutz- und Sicherheitscheckliste

- [x] keine oeffentliche Navigation fuer interne Seiten
- [x] interne Seiten aus Sitemap ausgeschlossen
- [x] interne Seiten mit `noindex,nofollow`
- [x] `robots.txt` sperrt `/intern/`
- [x] keine oeffentliche Registrierung vorgesehen
- [x] RLS fuer alle relevanten Tabellen aktiviert
- [x] privater Storage-Bucket vorgesehen
- [x] keine produktiven Zugangsdaten im Repository
- [x] lokale Zwischenspeicherung nur in IndexedDB
- [x] Fehlertexte ohne interne Details
- [x] keine Tracking- oder Marketing-Skripte im internen Bereich
- [ ] CSP fuer Produktivbetrieb mit Supabase-Domains pruefen und ergaenzen
- [ ] Benutzerkonten und Rollen produktiv anlegen und testen
